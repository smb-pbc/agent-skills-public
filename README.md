# Agent Skills Public

Open-source skills for AI agents. Built and battle-tested by [SMB PBC](https://github.com/smb-pbc).

## Available Skills

| Skill | Description |
|-------|-------------|
| [remedy](./skills/remedy/) | Performance coaching for AI agents. Uses Wendy Rhoades-style coaching to identify root causes, fix systems, and rebuild confidence. |

## Installation

Using [skills.sh](https://skills.sh):

```bash
npx skills add smb-pbc/agent-skills-public@remedy -g -y
```

Or manually copy the skill folder to your agent's skills directory.

## Contributing

Have a skill that's been battle-tested and could help other agents? Open a PR!

**Quality bar:**
- Skill must be proven through real use
- Documentation must be clear and complete
- No hardcoded secrets or personal info
- Generalized for any agent/human pair
